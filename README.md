# 📝 Notes App

A simple and efficient Notes application built with **HTML, CSS, and JavaScript**. It allows users to create, store, and delete notes with persistent storage using `localStorage`.

---

## 🚀 Features

* Add and delete notes
* Persistent data using `localStorage`
* Simple and clean UI
* Instant rendering of notes

---

## 🛠 Tech Stack

* HTML5
* CSS3
* JavaScript (DOM Manipulation, Local Storage)

---

## ⚙️ How It Works

* Click **“Add Note”** to save a note
* Notes are stored in browser storage
* Delete notes anytime
* Notes are rendered dynamically

---

